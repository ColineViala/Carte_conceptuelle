3D Mind Map Editor

The aim of this project is to create a 3D editor of concept maps (or semantic networks, or also called mind maps).

3D Mind Map. Why ?



Mind maps allow you to visualize a data system (for example: objects or ideas). It will facilitate understanding of the different links between the data in a clear, aesthetic and ergonomic way.

Advantages that they are in 3D are multiple, indeed it will allow the user to be able to move and therefore change his angle of view on his mind map. This third dimension will also make it possible to visualize more complex data sets.

3D Mind Map. For who ?


Concept maps in 3D can be used for any type of project. The primary utility of these visualization tools is educational. Indeed, they allow to realize the different links (hierarchical, phylogenetic, ...) between the data, to be able to appreciate a problem and thus to be able to find the best solution to solve them.

These 3D concept maps, beyond having an educational interest, can also have a professional interest. For example, to present a project to a client, usingthis kind of technology will allow the client to be more convinced and it will improving the image and reputation of the company that uses this kind of new technology.

3D Mind Map. What can we do?


With the 3D Mind Map Editor, you can :

  • - View the 3D concept map from all these angles in accordance with the movements of the mouse on the Web interface.

  • - Make modifications to the 3D concept map (move, rename, delete and create an object).

  • - Can save his concept map in 3D.

  • - Load one of his files and make a concept map of it.

  • - Customize his mind
